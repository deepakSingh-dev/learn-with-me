## About

`learn-with-me` is a running log of what I'm learning, written so future-me (and anyone following along) can actually **run it**, not just read it. Every notebook executes top-to-bottom, every code snippet is copy-pasteable, and every Git command has been tried in a real terminal.

No fluff, no half-finished examples — if it's in here, it works.
